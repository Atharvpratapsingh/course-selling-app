# CourseHub 🎓

A full-stack MERN course selling platform where instructors can publish courses and students can browse and purchase them.

## 🚀 Features

- 🔐 User authentication with JWT and bcrypt password hashing
- 👥 Role-based access control (Admin / User)
- 📚 Browse and view courses
- 💳 Mock payment flow with order creation and verification
- 📊 "My Courses" dashboard for purchased courses
- ⚙️ Admin panel to create new courses
- 🛡️ Protected routes with custom middleware

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, Tailwind CSS, React Router, Axios  
**Backend:** Node.js, Express.js, JWT, bcrypt  
**Database:** MongoDB Atlas (Cloud)  
**Tools:** Postman, Thunder Client, Git, GitHub, VS Code
